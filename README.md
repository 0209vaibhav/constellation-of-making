# Constellation of Making

**ideas, scattered and connected**

A 3D interactive data visualisation mapping a body of architectural and computational work. Every dot is a thought. Every cluster is a project.

---

## what it is

Rather than a list or a grid, this is a living field — where projects float as constellations of dots, each representing a fragment of work: its title, concept, methods, and ideas. Relationships between ideas matter as much as the projects themselves.

## how to explore it

- **Hover** over any dot to reveal its project and connected fragments
- **Filter** by layer — titles, subtitles, concepts, methods, and more
- **Rotate, zoom, and pan** to explore the field from any angle
- **Click** a highlighted dot to visit the project *(where available)*

## built with

- [Three.js](https://threejs.org/) — 3D rendering and instanced dot mesh
- [OrbitControls](https://threejs.org/docs/#examples/en/controls/OrbitControls) — rotate, zoom, and pan
- Vanilla JS — hover, raycasting, filter logic
- HTML & CSS — UI layer, tooltips, filters
- IBM Plex Mono + Inter — typography
